## ClashFX 1.0.0

### What's New

ClashFX is a modern macOS proxy client built on the mihomo core, designed for a smoother and more polished macOS experience.

**Highlights:**

- **Enhanced Mode (TUN)** — Full-device traffic proxy via TUN virtual network interface. Routes all system traffic (not just browser) through proxy rules, with one-click toggle from the menu bar
- **Built-in Config Editor** — Edit YAML configuration files with syntax highlighting, line numbers, and large file support. Includes Visual Mode for editing proxy nodes, groups, and rules through a form-based interface
- **5 Languages** — Full localization support: English, Simplified Chinese, Traditional Chinese, Japanese, and Russian. Switch anytime from the menu bar
- **Native macOS Integration** — Menu bar workflow, real-time speed indicator, system proxy management, keyboard shortcuts, and dark mode support
- **Remote Config Management** — Import and auto-update remote subscription links with smart refresh controls
- **Rule-Based Routing** — Full support for domain, IP-CIDR, GeoIP, process-based, and other Clash/mihomo routing rules
- **Multi-Protocol Support** — VMess, VLESS, Trojan, Shadowsocks, Hysteria2, WireGuard, and more

**System Requirements:** macOS 12.0+ (Monterey or later), Apple Silicon & Intel supported

### 新機能

ClashFX は mihomo コアに基づく最新の macOS プロキシクライアントです。より滑らかで洗練された macOS 体験を提供します。

**主な機能：**

- **拡張モード (TUN)** — TUN 仮想ネットワークインターフェースによるフルデバイストラフィックプロキシ。ブラウザだけでなく全てのシステムトラフィックをプロキシルールでルーティング
- **内蔵設定エディタ** — シンタックスハイライトと行番号対応の YAML エディタ。ビジュアルモードでプロキシノード、グループ、ルールをフォームから編集可能
- **5言語対応** — 英語、簡体字中国語、繁体字中国語、日本語、ロシア語に完全対応。メニューバーからいつでも切り替え可能
- **ネイティブ macOS 統合** — メニューバーワークフロー、リアルタイム速度表示、システムプロキシ管理、キーボードショートカット、ダークモード対応
- **リモート設定管理** — リモートサブスクリプションリンクのインポートと自動更新
- **ルールベースルーティング** — ドメイン、IP-CIDR、GeoIP、プロセスベースなど、Clash/mihomo ルーティングルールに完全対応
- **マルチプロトコル対応** — VMess、VLESS、Trojan、Shadowsocks、Hysteria2、WireGuard など

**システム要件：** macOS 12.0+（Monterey 以降）、Apple Silicon と Intel 対応

### Что нового

ClashFX — современный клиент прокси для macOS на базе ядра mihomo, разработанный для более плавного и качественного взаимодействия с macOS.

**Основные возможности:**

- **Расширенный режим (TUN)** — Полнодевайсное проксирование трафика через виртуальный сетевой интерфейс TUN. Весь системный трафик направляется через правила прокси
- **Встроенный редактор конфигурации** — Редактор YAML с подсветкой синтаксиса и номерами строк. Включает визуальный режим для редактирования прокси-узлов, групп и правил через формы
- **5 языков** — Полная поддержка: английский, упрощённый китайский, традиционный китайский, японский и русский. Переключение в любой момент из строки меню
- **Нативная интеграция с macOS** — Строка меню, отображение скорости в реальном времени, управление системным прокси, горячие клавиши, тёмная тема
- **Управление удалёнными конфигурациями** — Импорт и автообновление ссылок подписок
- **Маршрутизация по правилам** — Полная поддержка правил маршрутизации Clash/mihomo: домены, IP-CIDR, GeoIP, процессы
- **Поддержка множества протоколов** — VMess, VLESS, Trojan, Shadowsocks, Hysteria2, WireGuard и другие

**Системные требования:** macOS 12.0+ (Monterey или новее), поддержка Apple Silicon и Intel
